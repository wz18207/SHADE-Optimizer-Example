# SHADE-Optimizer-Example
3D Point Cloud SHADE Optimizer. Add SHADE+LM algorithm with a demo function.

# Environment

* Visual Studio 2017
* Kinect for Windows SDK v2.0
* Point Cloud Library v1.9.1
* .NET Core SDK 3.1.2(or later)
